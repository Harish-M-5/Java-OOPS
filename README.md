## üß© OOPs Concepts Explanation

- Class
Template used to create objects. Contains variables and methods.

- Object
Instance created from a class. Represents real-world entities.

- Encapsulation
Hiding data using private variables, providing access using getters and setters.

- Inheritance
One class (child) reuses the properties and methods of another class (parent).

- Polymorphism
Same method name behaving differently (method overloading and overriding).

### üìå Abstraction
Showing essential features and hiding the internal implementation using abstract classes & interfaces
